To test the app (Android)
-------------------------
Install ARCatalogue.apk from Apk folder in your Android device and run it.
Point with the camera to the Target_Maia.jpg and Target_prince.jpg images to see the AR (You can find them in Resources|Images or inside the project).

To build the project (Unity)
------------------------------
1- Login at https://www.easyar.com/ and and create a new license at https://portal.easyar.com/sdk/list 
2- Copy the license number inside the EasyARKey asset found at Assets|EasyAR|Common|Resources
3- Connect your device via USB (make sure your device allows USB debugging)
4- Press Ctrl+B or go to File|Build Settings|Build and Run
